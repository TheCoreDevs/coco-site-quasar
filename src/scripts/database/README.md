Put a json file into this directory to access the database + whitelist.
The project will not compile without this.

```json
{
  "whitelists": [],
  "firebase": {
    "apiKey": "",
    "authDomain": "",
    "databaseURL": "",
    "projectId": "",
    "storageBucket": "",
    "messagingSenderId": "",
    "appId": ""
  }
}
```
